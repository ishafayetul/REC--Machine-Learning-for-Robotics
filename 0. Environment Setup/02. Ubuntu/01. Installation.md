### Environment Setup before stating Machine Learning
#### On Ubuntu-
   1. Install Ubuntu latest version:
      - [Visit Ubuntu Official Guideline](https://ubuntu.com/tutorials/install-ubuntu-desktop#1-overview)
   2. Install VS Code (Must):
      ```
      sudo apt update
      sudo apt install wget gpg apt-transport-https software-properties-common -y
      wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor | sudo tee /usr/share/keyrings/packages.microsoft.gpg > /dev/null
      echo "deb [arch=amd64,arm64,armhf signed-by=/usr/share/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" | sudo tee /etc/apt/sources.list.d/vscode.list > /dev/null
      sudo apt update
      sudo apt install code
      ```
